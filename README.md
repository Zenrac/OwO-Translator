# OwO-Translator
## Installation
### Using pip
```
pip install TextToOwo
```
### Using github
```
pip install git+https://github.com/Zenrac/OwO-Translator
```
## Usage
```py
>>> from TextToOwO import owo
>>> owo.text_to_owo('This is a cool test !')

'This is a coow test ^w^'
```
